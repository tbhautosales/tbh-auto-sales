# TBH Auto Sales Website

## Deploy on Vercel
1. Go to Vercel and create a new project.
2. Upload this folder or import it from GitHub.
3. Click Deploy.
4. Add your domain `TBHautosales.com` in Project Settings > Domains.

## Update photos later
Replace these files in `/public`:
- `logo.png`
- `car1.jpeg`
- `car2.jpeg`
